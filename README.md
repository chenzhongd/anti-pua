## 🎯 项目宗旨

**拒绝画饼，把饼扔回老板脸上；拒绝内耗，把锅扣在老板头上。**

本项目致力于帮助职场人士识别、应对和反击各种职场 PUA 行为，维护自身合法权益，在职场中保持健康的心理状态和职业尊严。

**适合这些人阅读：**
- 被上级频繁否定、打压、自我怀疑的人
- 总被临时加活、职责外背锅的人
- 经常被“团队精神”“顾全大局”道德绑架的人
- 被反复画饼、承诺模糊、迟迟不兑现的人
- 已经开始考虑留证据、保饭碗、找后路的人

## 💡 核心准则

| 准则 | 说明 |
|------|------|
| **认清价值** | 你的价值不由他人定义，相信自己的能力 |
| **保持边界** | 合理的工作边界是健康的，不是自私的 |
| **证据为王** | 保留沟通记录，以备不时之需 |
| **及时止损** | 如果长期受压，考虑寻找新的机会 |
| **法律武器** | 严重侵权行为，学会用法律保护自己 |

## 📚 PUA 话术大全

### 1. 否定贬低型（摧毁自信）

这类话术通过持续打击你的个人价值，让你产生自我怀疑，从而不得不依赖公司。

- “你这点小事都做不好，还能指望你干什么？”
- “真不知道当初为什么会录用你。”
- “像你这种水平，换个地方早就被开除了，我是为你好才留着你。”
- “我原以为你很有潜力，现在你真的让我很失望。”
- “你看看别人，再看看你，差距怎么这么大？”
- “你这学历/经验，能来我们公司是你的福气。”

### 2. 情感绑架与制造焦虑

利用"机会"或"生存危机"作为诱饵，让你觉得现有的工作已经是最好的恩赐。

- “现在外面大环境这么差，找工作很难，你离开了公司根本活不下去。”
- “公司现在正处于关键期，大家都在加班，你怎么好意思准时下班？”
- “我对你已经是仁至义尽了，你要懂得知恩图报。”
- “大家都在拼命，就你一个人拖后腿，对得起团队吗？”
- “这个机会别人求都求不来，你要珍惜。”

### 3. 虚假承诺（画饼型）

通过灌输虚无缥缈的未来愿景，诱导你接受当下的低薪或超负荷工作。

- “年轻人不要动不动就提钱，要把眼光放长远，好的平台未来会带给你巨大收益。”
- “现在的辛苦是为了将来的磨炼，等这阵子忙完了，我一定不会亏待你。”
- “我是把你当接班人来培养的，这些超负荷的工作是对你的考验。”
- “等公司上市了，元老级员工都会有股权激励的。”
- “先付出后回报，这个道理你不懂吗？”

### 4. 道德绑架与牺牲论

将个人的私人利益或公司的剥削行为包装成"成长"或"团队精神"。

- “让你加班是给你快速成长的机会，我是为你好。”
- “你是新人，就应该多担待、多牺牲一点。”
- “大家都能完成任务，就剩你没做完，是不是态度有问题？”
- “年轻人要多学多干，不要只想着休息。”
- “团队利益大于个人利益，这点觉悟都没有？”

### 5. 模糊边界与责任转嫁

故意模糊工作范围，将额外工作包装成"分内之事"，出现问题时让你背锅。

- “这点小事顺手做一下就行了，不要分得那么清楚。”
- “既然你参与了这个问题，那就要承担相应责任。”
- “当时你也没表示反对，现在来追究有什么用？”
- “工作内容灵活一点，大家都是这么过来的。”
- “能者多劳嘛，这个交给你我放心。”

### 6. 孤立与排挤

通过边缘化、公开批评或制造信息差，让你感到被团队抛弃。

- “其他人都有意见，就不点名了，你自己心里应该有数。”
- “这个问题大家都在回避，你怎么这么爱表现？”
- “有人反映你不太合群，要注意一下。”
- “以后这种问题先私下沟通，不要在群里说。”
- “大家对你的印象不太好，要反思一下自己。”

### 打工人防 PUA 心法

别急着硬刚，先记住一个原则：**不在情绪里接招，只在规则里反杀。**

这一段负责讲**方法论**，具体怎么说，放到下面的话术区展开；这样就不会和后面的内容打架。

| 法则 | 核心思路 | 适用场景 |
|------|----------|----------|
| **别接情绪，先要事实** | 对方越想让你羞愧、慌张、自证，你越要把话题拉回具体问题 | 否定贬低、公开施压 |
| **别听画饼，只认落地** | 一切口头承诺，没写下来就等于没说过 | 画饼、升职加薪承诺 |
| **别被带节奏，先划边界** | 不争辩你“有没有觉悟”，只确认你“负责什么、做到什么程度” | 情感绑架、职责模糊 |
| **别靠感觉，凡事留痕** | 会议纪要、聊天截图、邮件确认，都是你未来的护城河 | 所有场景 |
| **别急着反驳，先让对方表态** | 让对方把不合理要求说完整、说明确，责任才更容易被固定 | 甩锅、越权、临时加码 |
| **别做老好人，要做有记录的人** | 能帮忙不等于无条件兜底，帮一次可以，长期吞下去就是默认 | 能者多劳、长期额外任务 |
| **别单打独斗，必要时升级** | 当问题已经不是沟通问题，而是制度或权益问题，就别只靠忍 | 持续施压、侵权、恶意针对 |

**一句话总结：**
- 对羞辱，不自证，先要细节
- 对画饼，不感动，只要书面
- 对甩锅，不背情绪，只分责任
- 对越界，不吵架，只谈规则
- 对长期压榨，不硬撑，边留证据边找后路

### 温和实战版

这部分主打：**话不难听，但边界明确；语气不撕破脸，但不会白吃亏。**

1. **当被要求无偿加班时：**
   > "我愿意配合公司紧急需求，但根据劳动法，加班应该给予调休或加班费。您看怎么处理比较合适？"

2. **当被否定价值时：**
   > "我很重视您的反馈。能否提供一个具体的改进计划，我可以按照这个目标努力？"

3. **当被画饼时：**
   > "我很期待公司的发展。能否将这些承诺写进劳动合同或补充协议里，让我更安心地投入工作？"

4. **当被孤立时：**
   > "如果团队合作有问题，我希望能开诚布公地讨论具体问题，而不是模糊的'大家都有意见'。"

### 💣 激进破防版（互联网复仇话术）

**警告：仅供娱乐和心理发泄之用，请勿在真实职场中使用！**

1. **当被否定价值时：**
   > "你算什么东西？拜托看清楚，我是来上班赚钱的，不是来被你洗脑的。真不知道当初哪根神经录用你做管理的。"

2. **当被要求无偿加班时：**
   > "加班？你怎么不问问你自己能给我多少钱？还是说你以为我是你妈，无限供应廉价劳动力？"

3. **当被画饼时：**
   > "呵呵，画饼画了三年了，能吃吗？我最喜欢的就是吃真实的工资，而不是你嘴里那虚无缥缈的前景。"

4. **当被道德绑架时：**
   > "你说团队精神？好啊，那我提议全公司统一薪资，这样大家都能体现团队精神。你敢同意吗？"

5. **当被孤立排挤时：**
   > "有意见就直说，别躲在'大家都这么说'后面。真要有问题，可以看录像回放，证据为王。"

6. **当被要求加班赶项目时：**
   > "你赶紧把我年薪翻倍写到合同里，然后我考虑要不要救你这个项目。要不然，你别在那儿指挥我。"

7. **当被强调"很失望"时：**
   > "我也很失望，失望你管理能力这么差还好意思装大尾巴狼。你失望没？没有啊？那咱俩都蛮好的。"

### 🐕 真实保命版（保护饭碗的生存秘笈）

**这才是你真正能用的！记住：永远不要让老板抓到把柄。**

1. **当被要求无偿加班时：**
   
   **心里想：** 老子加班是给你脸了？
   
   **嘴上说：** "好的，我记录一下今天的加班时间。按照公司规定，这个月的调休怎么安排呢？"（然后把邮件和时间戳保存好）

2. **当被否定价值时：**
   
   **心里想：** 你算老几？有种自己来做啊。
   
   **嘴上说：** "谢谢您的反馈，我理解您的期待很高。我现在的状态是……"（转向数据和具体工作内容，用事实说话，绝不自我检讨）

3. **当被画饼时：**
   
   **心里想：** 去你妈的美好未来，老子现在就要钱。
   
   **嘴上说：** "我很感谢公司给我的机会。关于您提到的……，能否后续以邮件形式确认一下细节？这样我能更好地规划职业发展。"（把承诺写进邮件里存档）

4. **当被道德绑架时：**
   
   **心里想：** 感恩？老子每个月都在用工作感谢你。
   
   **嘴上说：** "我会尽力做好工作。不过为了保证质量，我需要合理的工作周期。能否帮我理一下优先级？"（转向工作流程而不是心理层面）

5. **当被孤立排挤时：**
   
   **心里想：** 特么一群垃圾在那儿玩办公室政治。
   
   **嘴上说：** "感谢提醒，我会更加注意。如果有具体的反馈或建议，我很乐意直接沟通。"（继续做好工作，不参与八卦，保持专业）

6. **当被质问"你怎么这么计较"时：**
   
   **心里想：** 我就是计较，你咋地？
   
   **嘴上说：** "我觉得明确的流程和标准对大家都有好处，这样能避免后续的误会。"（把自我保护包装成"为了大家好"）

7. **当看出要被背锅时：**
   
   **心里想：** 没门儿，想甩锅给我？
   
   **嘴上说：** "这个事情我参与的部分是……，其他部分主要是……负责。建议我们一起回顾一下邮件记录，确保信息准确。"（及时查阅邮件和聊天记录，分清责任边界）

8. **当考虑要离职时：**
   
   **心里想：** 老子不干了，滚蛋。
   
   **嘴上说：** "感谢公司给我的机会和锻炼。我最近在思考职业方向……"（平稳过度，别撕破脸，保持专业形象，为下一份工作做准备。所有离职沟通都发邮件。）

**关键原则：**
- ✅ 所有承诺和指示都用邮件确认
- ✅ 定期备份自己的工作成果和邮件记录
- ✅ 遇事不决就找 HR 或发邮件留痕
- ✅ 永远礼貌但坚定，不给老板"找茬"的借口
- ✅ 工作和情绪分开，再生气也要笑脸相迎
- ✅ 一旦看出苗头不对，立即开启秘密求职模式
